# Junior's Help Desk — Frontend Prototype

A clickable prototype for first-day university navigation.

## Run
Open `index.html` directly, or use VS Code Live Server.

## Demo student
- Name: Ankit Junior

## Prototype admin
Open `admin.html` directly.

The admin page uses browser localStorage only. It is not a real shared database or secure admin system.

## Customize
Replace:
- `assets/university-building.jpg` with your university building photo.
- The `U` placeholders in HTML with an `<img>` using your university logo.
- University name, makers, portfolio links, locations and room routes with real data.

Class code convention used in this prototype:
`L / CAIT / PS / 405` = lecture type / subject abbreviation / professor abbreviation / room number.
